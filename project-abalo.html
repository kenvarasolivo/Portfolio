Architecture

FULL-STACK
ISLANDS ARCHITECTURE

An online marketplace for auto parts and tech products, built on Laravel with the Eloquent ORM for relational data. Rather than a full SPA, interactive React components are mounted as "Islands" inside server-rendered Blade templates — keeping pages fast and SEO-friendly while reserving client-side reactivity for exactly where it's needed.

INTERACTIVE
REACT FRONTEND

Built reusable React components driving shared state through the Context API — no prop drilling, no duplicated logic. Reactive elements like the Hero slider and a dynamic shopping cart update instantly, while a custom useEchoChannel hook abstracts away live-update wiring into a single, declarative interface.

LIVE NOTIFICATIONS

Event broadcasting powered by Laravel Reverb over a persistent WebSocket connection, consumed on the client via Laravel Echo. Public channels push global events like maintenance notices and new offers, while user-specific channels deliver targeted sale notifications — all without a single page refresh.
