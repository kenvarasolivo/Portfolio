01
Architecture

MODULAR
RESTFUL ARCHITECTURE

The existing codebase was a monolith — tightly coupled components that made every change risky. I refactored the entire backend into a modular RESTful architecture, separating concerns into independent service layers. Each endpoint became self-contained, testable, and easy to extend. The result: a codebase where adding a new lesson type or response mode no longer touched six different files.

02
Internationalization

DYNAMIC i18n
LANGUAGE SWITCHER

Students learn best in their native language. I built a dynamic internationalization system that lets users switch interface language and content locale on the fly — no page reload, no jarring transitions. The i18n engine loads translation packs from the REST API and hot-swaps all UI strings, making the chatbot instantly accessible to learners across Germany, Indonesia, and the English-speaking world.
